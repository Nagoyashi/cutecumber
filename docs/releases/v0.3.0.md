# v0.3.0 — Launch

cutecumber.cc goes public. After two production-hardening cycles in stealth
(`robots.txt` served `Disallow: /`), this release flips the switch: search
engines are now welcome, and the front door is held to the same accessibility
bar as everything behind it.

## Launch

- **Public launch.** `ROBOTS_ALLOW` is on — `robots.txt` now permits crawling.
  No page-weight or behavior change; the same zero-JS, zero-third-party,
  cookie-free public pages, now discoverable. (#35)

## Accessibility

- **Landing CTA meets WCAG AA.** The primary "claim your username" button was
  white-on-pink at 2.36:1 — below the 4.5:1 bar. It now uses a deep-cherry ink
  on the same pink (passing in both rest and hover states), and a regression
  test guards the brand-chrome contrast the way the preset themes are already
  guarded, so it can't quietly slip again. (#36)

## Docs

- Reconciled notes that lagged shipped reality: the Resend domain-verification
  caveat (the domain is verified; reset email works in production) and the
  resolved deploy-target decision. (part of #35)

## Notes

- No schema or API changes; the launch itself is a `fly.toml` flag flip +
  `fly deploy`.
- Growth work from the original Phase 5 sketch — paid decoration packs, more
  themes/avatars — is intentionally held for a later cycle: a payment processor
  is a new dependency and a new attack surface that earns its own written
  decision before it ships.

---

**Build story (raw material — owner supplies voice + facts):** the smallest
release by diff and the biggest by meaning — the day the stealth flag comes off.
Most of the work was making sure the front door was actually ready: a headless
Lighthouse pass on the landing page turned up one real defect — the primary call
to action, the single most important button on the site, failed WCAG AA contrast
(white text on a soft pink, 2.36:1). The brand chrome had never been contrast-
tested; only the user-page theme engine was. The fix keeps the pink and swaps the
label to a deep cherry ink that clears the bar in both states, plus a test so the
gap stays closed. The launch flip itself is one character in `fly.toml`.
